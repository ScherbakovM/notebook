# notebook

 Блокнот куда пользователь сможет ввести свои данные имя, фамилию, телефон  ( поля можно изменить при желании ).
 Каждому пользователю присваивается id.
 Доступные команды: CREATE - создать новго пользователя, READ - получить пользователя по id,
 LIST - получить список всех пользователей , DELETE- удалить кого-то пользователя по ид.
 Данные храняться в текстовом файле. 
 Логирование в консоли. 
 Для смены стиля форматирования документа нужно сменить методы в UserRepository saveAllOldFormat и readAllOldFormat на saveAllNewFormat и saveAllNewFormat (методы можно переписать при желании)
